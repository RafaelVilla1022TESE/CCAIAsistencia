Coloca en esta carpeta las dependencias nativas del SDK de huellas ZK9500 (ZKFinger).
Archivos comunes:
- zkfp.dll
- zkfpAlg.dll
- libzkfpcsharp.dll (segun arquitectura)
- data/dat, zkfpetk.dat u otros que entregue el SDK

El proyecto copia todo el contenido de la carpeta libs al directorio de salida al compilar.
